# DineSights — Restaurant Data Analysis Pipeline 🍽️
 ## Brief One Line Summary
An end-to-end restaurant data analysis project using SQL, Python (EDA & visualization), Excel, and Power BI to uncover sales, customer, and performance insights.
## Overview 📌
This project analyzes restaurant data to understand customer behavior, sales trends, and operational performance. Raw data was cleaned and queried using SQL, explored and visualized in Python (Pandas, Matplotlib/Seaborn), refined using Excel for manual checks and pivot summaries, and finally presented through an interactive Power BI dashboard for business-friendly insights.
## Problem Statement ⚠️
Restaurants generate large volumes of transactional and customer data but often lack a structured way to extract actionable insights. This project aims to answer questions like: Which items sell best? What are peak hours/days? How do ratings and revenue relate? Which locations or categories underperform?
